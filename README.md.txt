# Organização Solidária

Site de ONG fictícia com páginas:
- Início
- Projetos
- Cadastro

## Tecnologias usadas
- HTML5
- CSS3
- JavaScript (DOM, fetch, localStorage)

## Funcionalidades
- Navegação dinâmica com fetch
- Cadastro de usuários usando localStorage
- Modo escuro
- Responsividade
- Acessibilidade básica (WCAG 2.1)

## Como rodar
1. Clonar o repositório
2. Abrir `index.html` no navegador ou rodar com Live Server
